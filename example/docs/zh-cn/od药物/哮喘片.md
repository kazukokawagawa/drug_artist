# 哮喘片药品科普

## 一、主要作用
- 兴奋
- 心跳加速
- 躁狂
- 微弱的欣快感
- 眩晕

## 二、药品性状
本品为糖衣片，除去糖衣后，显棕黑色；味苦

## 三、主要成分
| 成分       | 备注                 |
|------------|----------------------|
| 罂粟壳     | 含阿片类活性成分     |
| 桔梗       | 传统止咳祛痰成分     |
| 麻黄       | 含麻黄碱类活性成分   |
| 甘草       | 调和药性，辅助祛痰   |

## 四、精神活性成分
- **吗啡**：阿片类中枢活性物质，具有镇痛、镇咳作用
- **麻黄碱**：中枢兴奋类物质，具有松弛支气管平滑肌作用

## 五、剂型与规格
- **剂型**：片剂
- **规格**：每片重0.35g（相当于原生药材1.062g）

## 六、推荐od剂量
18-40t
